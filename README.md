Metasploitable3 Lab (Student Edition)

This lab builds a vulnerable virtual machine used for cybersecurity training.

Time required:
30–60 minutes

Requirements:

* 16 GB RAM recommended
* 80 GB free disk space
* Stable internet connection

How to start:

1. Open the lab folder
2. Double-click Launch-BuildKit.bat
3. Follow the menu

If issues occur:

* Option 9 = Cleanup
* Option 10 = Full reset

Important:
This VM is intentionally vulnerable.
Use only in a safe lab environment.
